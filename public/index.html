增值税计算与缴纳测试题

一、单项选择题（每题2分，共20分）
1. （2分）
增值税是对商品生产、流通、劳务服务中多个环节的（ ）征收的一种流转税。
正确答案：B
增值税是对商品生产、流通、劳务服务中多个环节的新增价值或商品的附加值征收的一种流转税。
2. （2分）
小规模纳税人的年应征增值税销售额标准为（ ）万元及以下。
正确答案：C
小规模纳税人的标准为年应征增值税销售额500万元及以下。
3. （2分）
一般纳税人销售货物适用的基本税率是（ ）。
正确答案：C
13%税率适用于销售货物、提供加工、修理修配劳务以及进口货物等。
4. （2分）
建筑工程服务适用的增值税税率是（ ）。
正确答案：B
9%税率适用于销售交通运输、邮政、基础电信、建筑、不动产租赁服务，销售不动产，转让土地使用权等。
5. （2分）
现代服务业中的研发和技术服务适用的税率是（ ）。
正确答案：A
6%税率适用于销售服务、无形资产等，如现代服务业中的研发和技术服务、信息技术服务等。
6. （2分）
增值税的计税依据是（ ）。
正确答案：B
增值税的计税依据是销售额。销售额是指纳税人发生应税销售行为收取的全部价款和价外费用。
7. （2分）
一般纳税人增值税应纳税额的计算公式是（ ）。
正确答案：C
一般纳税人采用一般计税方法计算增值税，计算公式为：应纳税额 = 当期销项税额 - 当期进项税额。
8. （2分）
小规模纳税人适用简易计税方法，应纳税额 = （ ）。
正确答案：B
小规模纳税人适用简易计税方法，应纳税额 = 销售额×征收率。
9. （2分）
一般纳税人通常按（ ）进行纳税申报。
正确答案：A
一般纳税人通常按月进行纳税申报。
10. （2分）
纳税人出口货物适用（ ）税率。
正确答案：D
纳税人出口货物，税率为零；零税率并不意味着不征税，而是在出口环节不征收增值税，并且可以退还以前环节已缴纳的增值税。
二、判断题（每题1分，共10分）
11. （1分）
增值税是在商品或服务的全部销售额上征税。
正确答案：错误
增值税是对商品或服务的增值部分征税，而不是全部销售额。
12. （1分）
一般纳税人适用一般计税方法，可以抵扣进项税额。
正确答案：正确
一般纳税人适用一般计税方法，税率通常较高，但可以抵扣进项税额。
13. （1分）
小规模纳税人可以抵扣进项税额。
正确答案：错误
小规模纳税人适用简易计税方法，征收率相对较低，但不得抵扣进项税额。
14. （1分）
销售额是指纳税人发生应税销售行为收取的全部价款和价外费用，包括收取的销项税额。
正确答案：错误
销售额是指纳税人发生应税销售行为收取的全部价款和价外费用，但是不包括收取的销项税额。
15. （1分）
价外费用包括手续费、补贴、违约金、包装费等各种性质的价外收费。
正确答案：正确
价外费用包括价外向购买方收取的手续费、补贴、基金、集资费、返还利润、奖励费、违约金、滞纳金、延期付款利息、赔偿金、代收款项、代垫款项、包装费、包装物租金、储备费、优质费、运输装卸费以及其他各种性质的价外收费。
16. （1分）
企业将自产货物用于集体福利，不需要视同销售计算销售额。
正确答案：错误
企业将自产、委托加工的货物用于非增值税应税项目、集体福利或个人消费等，虽然没有实际的销售行为，但也要视同销售计算销售额。
17. （1分）
从销售方取得的增值税专用发票上注明的增值税额可以抵扣。
正确答案：正确
从销售方取得的增值税专用发票上注明的增值税额准予从销项税额中抵扣。
18. （1分）
用于集体福利的购进货物对应的进项税额可以抵扣。
正确答案：错误
用于简易计税方法计税项目、免征增值税项目、集体福利或者个人消费的购进货物、加工修理修配劳务、服务、无形资产和不动产，其进项税额不得从销项税额中抵扣。
19. （1分）
小规模纳税人一般按季度进行纳税申报。
正确答案：正确
小规模纳税人一般按季度进行纳税申报。
20. （1分）
一般纳税人可以自行开具增值税专用发票和普通发票。
正确答案：正确
一般纳税人可以自行开具增值税专用发票和普通发票；小规模纳税人通常只能自行开具增值税普通发票。
三、简答题（每题5分，共20分）
21. （5分）
请简述一般纳税人和小规模纳税人的主要区别。
参考答案：
1. 计税方法不同：一般纳税人适用一般计税方法；小规模纳税人适用简易计税方法。
2. 税率/征收率不同：一般纳税人税率较高（13%、9%、6%）；小规模纳税人征收率较低（3%）。
3. 进项税额抵扣：一般纳税人可以抵扣进项税额；小规模纳税人不得抵扣进项税额。
4. 销售额标准：一般纳税人年应征增值税销售额超过规定标准；小规模纳税人年应征增值税销售额500万元及以下。
22. （5分）
请列出一般纳税人适用的税率及其适用范围。
参考答案：
1. 13%税率：适用于销售货物、提供加工、修理修配劳务以及进口货物等。
2. 9%税率：适用于销售交通运输、邮政、基础电信、建筑、不动产租赁服务，销售不动产，转让土地使用权等。
3. 6%税率：适用于销售服务、无形资产等，如现代服务业中的研发和技术服务、信息技术服务等。
4. 零税率：适用于纳税人出口货物，境内单位和个人跨境销售国务院规定范围内的服务、无形资产。
23. （5分）
请列举准予从销项税额中抵扣的进项税额有哪些。
参考答案：
1. 从销售方取得的增值税专用发票上注明的增值税额。
2. 从海关取得的海关进口增值税专用缴款书上注明的增值税额。
3. 购进农产品，除取得增值税专用发票或者海关进口增值税专用缴款书外，按照农产品收购发票或者销售发票上注明的农产品买价和9%的扣除率计算的进项税额。
4. 从境外单位或者个人购进服务、无形资产或者不动产，自税务机关或者扣缴义务人取得的解缴税款的完税凭证上注明的增值税额。
24. （5分）
请简述增值税缴纳的基本流程。
参考答案：
1. 纳税申报：一般纳税人按月申报，小规模纳税人按季度申报，填写增值税纳税申报表。
2. 税款缴纳：申报成功后，按照规定期限缴纳税款，可以通过网上银行、办税服务厅等方式缴纳。
3. 发票管理：按规定开具、取得和保管增值税专用发票和普通发票。
4. 税务检查与风险防控：做好自查自纠，及时发现和纠正税务风险，关注税收政策变化。
四、计算题（每题10分，共30分）
25. （10分）
一般纳税人计算题

某生产企业为一般纳税人，20XX年X月发生以下业务：
1. 购进原材料一批，取得增值税专用发票，注明价款200万元，增值税税额26万元，款项已支付。
2. 销售产品一批，开具增值税专用发票，销售额为400万元，增值税税额52万元，款项已收到。
3. 当月发生运输费用15万元，取得运输公司开具的增值税专用发票，注明增值税税额1.35万元。

要求：计算该企业当月应缴纳的增值税。
参考答案：
进项税额 = 26 + 1.35 = 27.35万元
销项税额 = 52万元
应纳税额 = 52 - 27.35 = 24.65万元
26. （10分）
小规模纳税人计算题

某小规模纳税人20XX年第二季度发生以下业务：
销售货物取得含税收入103万元（征收率3%）。

要求：计算该小规模纳税人第二季度应缴纳的增值税。
参考答案：
不含税销售额 = 103 ÷ (1 + 3%) = 100万元
应纳税额 = 100 × 3% = 3万元
27. （10分）
综合计算题

某贸易公司为一般纳税人，20XX年X月发生以下业务：
1. 购进商品一批，取得增值税专用发票，注明价款300万元，增值税税额39万元。
2. 销售商品一批，开具增值税专用发票，销售额为500万元，增值税税额65万元。
3. 将购进的一批商品用于职工福利，该批商品购进价款50万元，增值税税额6.5万元。

要求：计算该企业当月应缴纳的增值税。
参考答案：
可抵扣的进项税额 = 39 - 6.5 = 32.5万元（用于职工福利的进项税额不得抵扣）
销项税额 = 65万元
应纳税额 = 65 - 32.5 = 32.5万元

28. （10分）
一般纳税人会计分录

某一般纳税人发生以下业务：
1. 采购一批原材料，价款150万元，增值税税额19.5万元，款项已通过银行支付。
2. 销售一批产品，售价300万元，增值税税额39万元，款项收到存入银行。
3. 月末结转本月应交未交增值税19.5万元。

要求：编制上述业务的会计分录。
参考答案：
1. 采购原材料时：
借：原材料 150
    应交税费——应交增值税（进项税额） 19.5
    贷：银行存款 169.5

2. 销售产品时：
借：银行存款 339
    贷：主营业务收入 300
        应交税费——应交增值税（销项税额） 39

3. 月末结转增值税：
借：应交税费——应交增值税（转出未交增值税） 19.5
    贷：应交税费——未交增值税 19.5
